Página de registro (signup.html)

En la página de registro se ha utilizado un formulario, concretamente el que se puede ver en el extracto de código que se muestra a continuación. En este formulario se han utilizado las etiquetas required en tres campos para hacer que sean cubiertos de forma obligatoria: contraseña, correo electrónico, nombre, fecha de nacimiento y términos y condiciones.

Para cada campo, además, se ha utilizado el tipo conveniente, para un adecuado control.

  • Contraseña: password
  • Correo: email
  • Fecha de nacimiento: date
  • Teléfono: tel
  • Resto: text
